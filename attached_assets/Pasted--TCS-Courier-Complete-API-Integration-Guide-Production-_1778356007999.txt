# IMPORTANT — TCS Uses 2 Different Tokens

TCS authentication works in TWO steps.

## STEP 1

Generate Bearer Token

## STEP 2

Generate Access Token

These tokens are DIFFERENT.

Never overwrite one token with another.

# COMPLETE AUTHENTICATION FLOW

## STEP 1 — Generate Bearer Token

### Laravel Example

```php
public function generateBearerToken()
{
    $url = 'https://ociconnect.tcscourier.com/auth/api/auth';

    $response = Http::post($url, [
        'clientid' => env('TCS_CLIENT_ID'),
        'clientsecret' => env('TCS_CLIENT_SECRET')
    ]);

    if (!$response->successful()) {
        throw new Exception('Bearer Token Failed: ' . $response->body());
    }

    $data = $response->json();

    return $data['accessToken'];
}
```

---

# STEP 2 — Generate Access Token

```php
public function generateAccessToken($bearerToken)
{
    $url = 'https://ociconnect.tcscourier.com/ecom/api/authentication/token';

    $response = Http::withHeaders([
        'Authorization' => 'Bearer ' . $bearerToken,
        'Content-Type' => 'application/json'
    ])->post($url, [
        'username' => env('TCS_USERNAME'),
        'password' => env('TCS_PASSWORD')
    ]);

    if (!$response->successful()) {
        throw new Exception('Access Token Failed: ' . $response->body());
    }

    $data = $response->json();

    return $data['accesstoken'];
}
```

---

# IMPORTANT TOKEN LOGIC

```php
$bearerToken = generateBearerToken();
$accessToken = generateAccessToken($bearerToken);
```

Never do:

```php
$accessToken = $bearerToken;
```

This causes:

* HTTP 403
* Invalid access token
* Booking failures

# BOOK SHIPMENT API

```php
public function bookShipment($order)
{
    $token = $this->getAccessToken();

    $url = 'https://ociconnect.tcscourier.com/ecom/api/shipment/book';

    $payload = [
        'consigneeName' => $order->customer_name,
        'consigneeAddress' => $order->address,
        'consigneeMobNo' => $order->phone,
        'shipperName' => 'Khan Dry Fruits',
        'weightInKg' => number_format($order->weight, 1),
        'pieces' => 1,
        'codAmount' => $order->total,
        'orderReferenceNo' => $order->id,
        'serviceType' => 'OVERNIGHT',
        'originCityName' => 'Lahore',
        'destinationCityName' => $order->city
    ];

    $response = Http::withHeaders([
        'Authorization' => 'Bearer ' . $token,
        'Content-Type' => 'application/json'
    ])->post($url, $payload);

    if (!$response->successful()) {
        Log::error('TCS Booking Error', [
            'status' => $response->status(),
            'body' => $response->body()
        ]);

        throw new Exception($response->body());
    }

    return $response->json();
}
```

# WEIGHT FIX

TCS requires decimal weight.

Correct:

```php
0.5
1.0
2.5
```

Wrong:

```php
1
null
''
```

Always use:

```php
number_format($weight, 1)
```
